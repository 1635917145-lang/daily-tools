# 🔧 daily-tools 个人日常小工具集合
大学生自用Python效率工具 | 学习练手项目

## 📌 项目介绍
利用课余时间编写的轻量化日常工具，解决学习和生活中的小问题，
巩固Python基础语法，践行学以致用的学习理念。

## ✨ 包含工具
1. 📂 **文件自动分类工具**
   自动整理桌面/下载文件夹，按图片、文档、代码、压缩包分类
2. 📝 **个人待办清单**
   学习计划管理，支持添加、查看、标记完成，数据持久化存储
3. 💰 **日常记账工具**
   记录生活费支出，自动统计总花费，适合大学生财务管理
